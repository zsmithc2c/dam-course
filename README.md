# DAM Course — Data Analysis & Management

**Code2College | HireWheel**

Notebooks and datasets for the Data Analysis & Management (DAM) async course.

## Structure

```
notebooks/
  phase_1/   — Python & NumPy foundations (L1, L2, L4, L5)
  phase_2/   — SQL mini-project (L9 optional supplement)
  phase_3/   — Data analysis & visualization (L11-L14)
  phase_4/   — Capstone project (L15-L16)
datasets/
  phase_1/   — employee_data, healthcare_sample, student_grades, country_stats
  phase_2/   — fast_food_nutrition (clean + messy), pokemon, pokemon_evolutions
  phase_3/   — netflix_titles, tx_accidents_2022_sample
```

## Opening Notebooks in Colab
Click any notebook link in HireWheel, or open directly via Google Colab.
